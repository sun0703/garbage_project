### 3.2 AI 推理架构（核心亮点）

系统支持三种推理架构，按优先级自动选择：

**优先级 1：最优双层架构 (optimal_dual_layer_v2)**
```
Layer 1: V2 粗分类器 (best v2.pt, 12类→4大类, 1280×1280 高分辨率)
Layer 2: 主模型精细分类 (garbage_yolov8m_best.pt, 40类)
Layer 2.5: SAHI 切片增强 (320×320 切片)
Smart Fusion: V2权重0.45 + 主模型0.40 + SAHI 0.15
```

**优先级 2：标准三层架构 (standard_triple_layer)**
```
Layer 1: YOLOv8 目标检测 → 粗定位 + 大类分类
Layer 2: SAHI 切片推理 → 小目标增强检测
Layer 3: 双层级联 (粗分类→路由→专用子模型) → 精细识别
Fusion: 加权投票 + 置信度校准 → 最终决策
```

**优先级 3：单模型 + 特征降级**
```
YOLOv8 单模型推理 → 40类映射
Fallback: 12维图像特征启发式分类 (颜色/纹理/边缘/金属光泽)
```

### 3.3 降级策略

| 场景 | 策略 |
|------|------|
| 专用模型不可用 | 自动降级到 COCO 80类 → 中国4类映射 |
| 所有模型不可用 | 12维图像特征启发式分类（演示模式） |
| SAHI 不可用 | 跳过切片推理，仅用 YOLO + 级联 |
| 缓存命中 | 直接返回，不调用模型（<50ms） |

```
DatabaseBackend (抽象基类)
├── SQLiteDatabase     ← 开发环境默认，零配置
│   └── WAL 模式 + 外键约束
└── PostgreSQLDatabase ← 生产环境，Docker Compose 自动注入
```

## 八、安全与运维

### 8.1 限流策略

| 路径 | 限制 | 说明 |
|------|------|------|
| /api/predict | 20 次/分钟 | 计算密集型 |
| /api/batch_predict | 10 次/分钟 | 批量推理 |
| /api/search | 30 次/分钟 | 轻量查询 |
| /api/auth/* | 10 次/分钟 | 防暴力破解 |
| 默认 | 30 次/分钟 | 全局兜底 |

限流响应头：`X-RateLimit-Limit` / `X-RateLimit-Remaining` / `X-RateLimit-Reset` / `Retry-After`

### 8.2 缓存策略

| 策略 | 参数 | 说明 |
|------|------|------|
| LRU 淘汰 | max_size=500 | 超容量淘汰最久未使用 |
| TTL 过期 | ttl=24h | 过期自动删除 |
| 感知哈希去重 | phash (hash_size=16) | 相似图片命中同一缓存 |

### 8.3 认证体系

| 方式 | 实现 | 说明 |
|------|------|------|
| 用户名密码 | SHA-256 哈希 | 注册/登录/注销 |
| OAuth | 微信 + GitHub | 预留接口，需配置 App ID/Secret |
| Session | 服务端存储 | 有效期 7 天 |
| 管理员 | 独立登录体系 | Session Cookie 鉴权，密码从环境变量或自动生成 |

### 8.4 监控

- **Prometheus 指标**: `/api/metrics` 端点，自定义业务指标
- **健康检查**: `/api/health` 检查模型和服务状态
- **日志**: 结构化 JSON 日志，RotatingFileHandler（10MB×5份）

## 十四、关键设计决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 前端框架 | 原生 JS SPA | 零构建依赖、单人开发友好、可渐进迁移到 Vue3 |
| 数据库 | SQLite/PG 双模式 | 开发零配置、生产可扩展 |
| AI 架构 | 三层融合 + 双层可选 | 多视角交叉验证提升准确率 |
| 认证方案 | Session Cookie | 简单可靠、适合服务端渲染场景 |
| 限流方案 | 自实现滑动窗口 | 零外部依赖、路径分级灵活 |
| 缓存方案 | LRU+TTL+phash | 感知哈希去重避免重复推理 |
| 管理后台 | SPA 内嵌 /admin | 一套代码维护、共享 API 和认证 |
| 公网访问 | Cloudflare Tunnel | 免费、自动 HTTPS、无需公网 IP |
