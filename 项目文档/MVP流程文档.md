# 校园生活垃圾智能分类识别系统 - MVP流程文档

## 1. MVP概述

### 1.1 MVP定义

MVP（Minimum Viable Product，最小可行产品）是指具备核心功能、能够验证产品价值、且可以快速迭代的最小版本。

### 1.2 MVP目标

本期MVP版本的核心目标是：

| 目标 | 指标 | 优先级 |
|------|------|--------|
| 功能可用 | 完成拍照→识别→展示完整流程 | P0 |
| 性能达标 | 识别速度 ≤500ms，准确率 ≥85% | P0 |
| 体验友好 | 界面简洁，操作步骤少 | P0 |
| 可扩展 | 架构清晰，便于后续功能扩展 | P1 |

### 1.3 MVP范围

**包含功能**：
- ✅ 拍照识别（本地模型推理）
- ✅ 4类垃圾分类
- ✅ 结果展示与投放指引
- ✅ 识别结果反馈

**不包含功能**（后续版本）：
- ⬜ 实时摄像头识别
- ⬜ 历史识别记录
- ⬜ 离线模式切换
- ⬜ 多语言支持

### 2.2 流程详细说明

#### 2.2.1 启动与初始化

**步骤1：启动应用**

| 操作 | 用户行为 | 系统响应 |
|------|----------|----------|
| 1.1 | 打开APP | 显示启动页/加载动画 |
| 1.2 | - | 后台加载TFLite模型 |
| 1.3 | - | 模型加载完成 |
| 1.4 | - | 跳转到首页 |

**预期耗时**：首次启动 2-3秒，后续启动 <1秒

#### 2.2.2 拍照识别

**步骤2：拍照**

| 操作 | 用户行为 | 系统响应 |
|------|----------|----------|
| 2.1 | 点击"拍照识别"按钮 | 打开系统相机 |
| 2.2 | 拍摄照片 | 保存临时图片 |
| 2.3 | 点击"确认" | 关闭相机，进入预处理 |
| - | 点击"重拍" | 返回相机继续拍摄 |

**步骤3：预处理**

| 操作 | 用户行为 | 系统响应 |
|------|----------|----------|
| 3.1 | - | 读取图片文件 |
| 3.2 | - | 调整图像尺寸为224×224 |
| 3.3 | - | 归一化像素值到[0,1] |
| 3.4 | - | 标准化（ImageNet参数） |
| 3.5 | - | 转换为模型输入张量 |

**步骤4：模型推理**

| 操作 | 用户行为 | 系统响应 |
|------|----------|----------|
| 4.1 | - | 将预处理结果输入模型 |
| 4.2 | - | 执行推理计算 |
| 4.3 | - | 获取输出logits |
| 4.4 | - | Softmax转换为概率 |
| 4.5 | - | 选取最大概率类别 |

**步骤5：结果展示**

| 操作 | 用户行为 | 系统响应 |
|------|----------|----------|
| 5.1 | - | 映射类别ID到类别信息 |
| 5.2 | - | 格式化置信度显示 |
| 5.3 | - | 生成投放指引文本 |
| 5.4 | - | 展示结果页面 |

## 5. 数据流程

### 5.1 识别数据流

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│   摄像头  │ → │  原始图片 │ → │  预处理   │ → │  模型输入 │ → │  模型推理 │
│  (Camera) │    │  (JPEG)  │    │  (张量)  │    │  张量     │    │  (TFLite)│
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └────┬─────┘
                                                                         │
                                                                         ▼
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│   用户   │ ← │  结果展示 │ ← │  结果处理 │ ← │  概率输出 │ ← │  输出张量 │
│          │    │  (UI)    │    │           │    │  Softmax │    │          │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
```

### 5.2 数据处理节点

| 节点 | 输入 | 输出 | 处理时间 |
|------|------|------|----------|
| 图像采集 | - | JPEG图片 | 实时 |
| 图像预处理 | JPEG (800KB) | Float32List (600KB) | ~50ms |
| 模型推理 | Float32List | Float32List (4元素) | ~100ms |
| 后处理 | logits | PredictionResult | ~10ms |
