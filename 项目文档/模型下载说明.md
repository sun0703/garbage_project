## 📥 下载方式

### 方式一：从官方源下载（推荐）

```bash
# 1. 创建 models 目录（如果不存在）
mkdir models

# 2. 使用 wget 下载模型文件
wget -O models/garbage_yolov8m_best.pt "https://your-model-source.com/garbage_yolov8m_best.pt"
```

### 方式二：使用 Python 脚本下载

```python
import urllib.request
import os
from pathlib import Path

# 模型下载 URL（请替换为实际地址）
MODEL_URL = "https://your-model-source.com/garbage_yolov8m_best.pt"
MODEL_PATH = Path("models/garbage_yolov8m_best.pt")

# 创建目录
os.makedirs("models", exist_ok=True)

# 下载模型
print("正在下载模型文件...")
urllib.request.urlretrieve(MODEL_URL, MODEL_PATH)
print(f"✅ 模型已下载到: {MODEL_PATH}")
```

### 方式三：手动下载

1. 访问模型托管地址（请联系项目维护者获取链接）
2. 下载 `garbage_yolov8m_best.pt` 文件
3. 在项目根目录创建 `models` 文件夹
4. 将模型文件放入 `models/` 目录

## 🔧 故障排除

### 问题1：程序启动时提示"模型文件不存在"

**原因**：模型文件未下载或路径不正确

**解决方案**：
1. 确认 `models/` 目录存在
2. 确认模型文件名完全匹配：`garbage_yolov8m_best.pt`
3. 检查文件路径是否在项目根目录下（与 `garbage_project/` 同级）

### 问题2：下载速度很慢

**建议**：
- 使用下载工具（如 IDM、迅雷）加速
- 尝试更换网络环境
- 联系项目维护者获取镜像下载地址

### 问题3：模型文件损坏

**症状**：程序报错 `RuntimeError: invalid magic number`

**解决方案**：
1. 删除现有模型文件
2. 重新下载
3. 校验文件完整性（如提供 MD5/SHA256）
